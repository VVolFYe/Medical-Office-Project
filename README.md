# Medical-Office-Project
- Project created by Tudor Boscu and Florin-Andrei Docheru

# Description:
- The project implements a Medical Doctor's Office and multiple feature regarding this. We decided to post the scrpts and explanations regarding what we did on GitHub to document our work through this small project.

# Features:
- View all doctors from the database
- View appointments grouped by doctor using a master-detail page
- Search a patient's appointments by CNP
- Display doctor statistics, including the number of appointments for each doctor

# For this project we used:
- Oracle APEX
- Oracle SQL
- SQL Scripts
